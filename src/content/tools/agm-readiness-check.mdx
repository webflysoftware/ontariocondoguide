## How to use

Check each item as you complete it. The tool calculates a readiness score and highlights missing steps. Pair it with the full [AGM checklist](/templates/agm-checklist) and the [Ontario condo AGM guide](/guides/ontario-condo-agm-guide).

## When to run this check

Run the readiness check **at least 6–8 weeks before your target AGM date** — earlier if elections, special business, or electronic voting are involved. Re-run it after notice is sent to confirm proxy collection and quorum tracking are on track.
